Moderation Kitchen

Moderation kitchen consists of a web application built using the Angular framework, and a web API built using ASP.NET Core. These are both hosted in Azure.
The web API is responsible for storing and retrieving recipe data from a persistent store.
Persistence is implemented using simple JSON files as this was a easier, hassle free way of storing this data.
The number of recipes is expected to be relatively low so performance was not a concern for this application.
