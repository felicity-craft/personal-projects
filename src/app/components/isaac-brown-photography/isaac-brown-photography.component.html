Isaac Brown Photography

Isaac Brown Photography is a website for a photographer. It consists of a web application which shows available prints, these prints can be added to, or removed from a cart, and items in the cart can be purchased.
The web application is implemented using Angular.
Print related data is stored in a JSON file which is served by the web server which hosts the application.
Cart related data is stored in the browser's local storage.
